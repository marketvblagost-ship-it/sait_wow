# AmalSite — финальная версия для GitHub Pages

## Что внутри
- Реальные PNG-мокапы в hero и портфолио.
- Цена: 9 000 ₽ зачёркнута, 4 499 ₽ до 10 мая выделена.
- Все кнопки «Получить макет» открывают квиз.
- BotFAQtor и Яндекс.Метрика сохранены в `<head>`.
- Квиз готов к подключению webhook через `LEAD_ENDPOINT` в `script.js`.

## Как загрузить
Содержимое архива загрузить в корень репозитория GitHub Pages:
- `index.html`
- `style.css`
- `script.js`
- папка `assets/`

Не загружать папку-обёртку, только содержимое архива.

## Проверка после загрузки
1. Открыть сайт.
2. Проверить первый экран на ПК и телефоне.
3. Нажать «Получить макет».
4. Пройти квиз.
5. Проверить, что открывается Telegram и/или webhook отправляет заявку.
6. Проверить цели в Яндекс.Метрике.


## Обновление скорости
Изображения hero и портфолио сжаты в WebP. Это снижает вес сайта и ускоряет загрузку на GitHub Pages.
